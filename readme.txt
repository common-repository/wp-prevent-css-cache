WordPress plugin to prevent caching your stylesheet during development

== Description ==

This WordPress Plugin prevents the browser from caching your main stylesheet during development. This comes in handy when you have clients and others viewing your work and you don't want to explain to them that they need to clear their browser's cache all the time.


Major features in WP Enable WebP include:

* No Setup Required
* Doesn't touch the database

== Installation ==

Upload the "wp-prevent-css-cache" folder to your wp-content/plugins directory, then Activate it.
